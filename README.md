# NBA Roster & Cap Board

NBA全30チームのロスター、背番号、選手サラリー、チーム総キャップ、エプロン判定、トレード／異動をまとめて見るローカルダッシュボードです。

## 起動

`start.cmd` をダブルクリックすると、最新データを取得してブラウザで開きます。

## 自動更新

起動中は6時間ごとに自動更新します。画面右上の「データを更新」でも即時更新できます。

PCを閉じている間も毎朝更新したい場合は、PowerShellを管理者として開いて `install-daily-update.ps1` を実行してください。Windowsタスクスケジューラへ毎日午前6時の更新を登録します。

## データについて

- ロスター、背番号、選手契約サラリー、異動: ESPN公開エンドポイント
- チーム総キャップ、デッドキャップ、エプロン基準: Spotrac Team Cap Tracker
- ESPNのtransactionsには契約、ウェイブ、スタッフ異動なども含まれるため、画面では文面から「トレード」を分類しています。
- データ提供元の仕様変更時は `src/update.mjs` の取得アダプターを調整してください。

契約・CBA判断の最終確認にはNBA公式発表、CBA、各チーム発表も参照してください。

## インターネット公開

v1.0はGitHub Pages公開に対応しています。`.github/workflows/deploy-pages.yml` がサイト生成と毎日のデータ更新を行います。詳しい手順は `outputs/PUBLISHING_GUIDE.md` を参照してください。
